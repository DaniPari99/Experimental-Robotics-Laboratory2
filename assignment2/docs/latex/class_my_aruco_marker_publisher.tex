\doxysubsection{Detailed Description}
counter to count the seconds for temporizing the arm motion 

A class for detecting markers with the camera and for publishing informations about rooms.

This class detects the Aruco markers with the camera, asks for room informations embedded in them through the {\ttfamily /info\+\_\+room} service (provided by {\ttfamily marker\+\_\+server}) and publish those informations in the {\ttfamily info/room} topic.
